# LOLDanJang (LoL Esports Simulator)

이 프로젝트는 거대언어모델(LLM)인 Gemini를 기반으로 작동하는 텍스트 기반 경영 시뮬레이터입니다.

사용자가 가상의 LoL 프로 게임단 단장이 되어, 2026년 시즌을 운영하는 과정을 다룹니다.

### 📝 프로젝트 소개

기존의 단순 대화형 프롬프트를 기반으로 하되, 보다 일관성 있는 시뮬레이션을 위해 논리 구조를 보완했습니다.

LLM이 게임 내 상황을 더 정확히 판단할 수 있도록, 자연어 처리(NLP)에 최적화된 규칙을 적용했습니다.

### ⚙️ 주요 기능 및 로직

**1. 데이터 서식 통일**

선수 명단(Roster)과 팀 재정 상태를 정해진 텍스트 양식(Format)으로 정의하여, 시스템이 데이터를 유지하도록 했습니다.

**2. 경기 진행 로직**

승패가 단순히 무작위로 결정되지 않도록, 팀 전력과 컨디션 변수를 고려하는 내부 규칙을 마련했습니다.

**3. 시즌 운영 시뮬레이션**

스토브리그 영입부터 정규 시즌, 월즈(Worlds)까지 이어지는 일련의 과정을 텍스트로 구현했습니다.

### 🛠️ 사용 환경 및 가이드

이 시뮬레이터는 Google Gemini 1.5 Pro 모델(Thinking Mode)에서 가장 안정적으로 작동합니다.

저장소에 포함된 `.md` 파일(규칙)과 로스터 파일을 Gemini 채팅창에 업로드하여 실행합니다.

빠른 응답 모드보다는 논리적 추론이 가능한 모드를 사용할 것을 권장합니다.

### ℹ️ 출처 및 참고 사항

본 프로젝트의 기초 아이디어와 원본 프롬프트는 '아카라이브 리그 오브 레전드 채널'의 게시물을 참고했습니다.

선수들의 실제 데이터는 통계 사이트 'gol.gg'를 참조 기준으로 삼고 있습니다.

현재 깃허브 업로더는 원작자가 아니며, 원본 프롬프트의 로직 개선 및 최적화 작업을 수행했습니다.
